== Description ==

Questo plugin di permette di creare in modo rapido e veloce i DDT (documento di trasporto) per il corriere BRT dagli ordini in Woocommerce o in modalità "Libera".
 
Quando si effettua una consegna dei prodotti venduti ad un cliente, questa deve essere sempre accompagnata da un DDT, ovvero un documento di ritiro.

Con l’installazione del plugin DDT BRT FREE potrai creare il DDT per il corriere BRT direttamente sul tuo e-commerce (Woocommerce), utilizzando i dati del cliente per la sezione "Destinatario", velocizzando in questo modo la compilazione del documento. Se presente la modalità di pagamento "Contrassegno", il sistema compilerà in modo automatico la sezione inerente.

Con il Plugin DDT BRT FREE potrai:

1. Creare il DDT in formato PDF dal tuo Woocommerce (Limite 4 al giorno);
2. Possibilità di creare DDT Liberi (non vincolati ad un ordine di Woocommerce) ;
3. Utilizzare i dati destinatario inseriti dal cliente al momento dell’ordine;
4. Possibilità di scaricare o stampare il DDT in formato PDF dal tuo browser;
5. Inserimento di un mittente fisso;

Con il Plugin DDT BRT PRO potrai:

1. Creare il DDT in formato PDF dal tuo Woocommerce (Nessun limite giornaliero) ;
2. Possibilità di creare DDT Liberi (non vincolati ad un ordine di Woocommerce) ;
3. Utilizzare i dati destinatario inseriti dal cliente al momento dell’ordine;
4. Possibilità di scaricare o stampare il DDT in formato PDF dal tuo browser;
5. Possibilità di avere più Mittenti inseriti in archivio e selezionare quello opportuno nella fase di creazione DDT;
6. Possibilità di generare il Tracking Code tramite l’inserimento del numero dell’etichetta rossa BRT;
7. Se il Tracking Code è abilitato, il link per seguire la spedizione verrà inviato all’utente nell’email “Ordine Completato”;
8. Archivio storico dei DDT emessi;
9. Possibilità di modifica, visualizzazione ed eliminazione dei DDT già creati;

Puoi acquistare DDT BRT PRO qua:
[Link qui](https://www.webalchlab.it/prodotto/plugin-wordpress/plugin-ddt-brt-pro-per-woocommerce)

 
== Installation ==

Puoi utilizzare la funzione "Aggiungi plugin" interna a Wordpress, Cercando "WebAlchLab DDT BRT", oppure scarica lo zip:

1. Unzippa il file sul tuo computer in locale.
2. Carica la cartelle del plugin all’interno di `/wp-content/plugins/`
3. Attiva il plugin attraverso la sezione ‘Plugins’ di WordPress.
 
== Frequently Asked Questions ==
  
= Il plugin non funziona. Perché ? =

Prima di tutto verifica di aver scaricato l’ultima versione di Wordpress, l’ultima versione di Woocommerce e l’ultimo aggiornamento del plugin.
Se il problema persiste, prova a disattivare e riattivare il plugin.

= Ho un problema con il plugin, a chi posso chiedere? =
 
1. Puoi compilare il [form contatti](https://www.webalchlab.it/contatti) ed inviare la segnalazione del problema, bug o suggerimenti per gli aggiornamenti futuri.
